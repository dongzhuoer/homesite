2 Create repo

You can create a repo by directly push a local repo to GitLab:

  1. open empty,

  2. export repo=repo_name

  3. push to GitLab using following code

    git remote add gitlab git@gitlab.com:dongzhuoer/$repo.gitbook.git
git push -u gitlab master
git remote remove gitlab
